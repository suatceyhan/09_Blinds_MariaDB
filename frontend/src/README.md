# `src`

Uygulama kaynağı. Katmanlar:

- **`app`** — kabuk, router, layout, global provider’lar.
- **`features`** — iş özellikleri (sayfalar, feature API client’ları).
- **`entities`** — tekrar kullanılan domain parçaları (liste satırı, küçük kartlar).
- **`widgets`** — birden fazla entity birleşen UI blokları.
- **`processes`** — sihirbaz / çok adım akışları.
- **`shared`** — tasarım sistemi, `lib`, hook’lar.
