# API sözleşmesi (contracts)

- **Amaç:** Web, mobil ve harici tüketiciler için **tek API sözleşmesi** (OpenAPI 3.x).
- **Kaynak:** Elle yazılan YAML veya FastAPI’den export (`openapi.json`).
- **Kullanım:** Kod üretimi (client SDK), contract testleri, sürümleme (v1/v2 ayrı dosya olabilir).

Bu klasör şimdilik boş; backend ayağa kalkınca export buraya kopyalanabilir.
